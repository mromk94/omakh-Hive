## 🚀 **HOW TO USE**

### **Access the Dashboard:**

1. **Start Backend:**
   ```bash
   cd backend/queen-ai
   source venv/bin/activate
   uvicorn app.main:app --reload
   ```

2. **Start Frontend:**
   ```bash
   cd omk-frontend
   npm run dev  # Port 3001
   ```

3. **Navigate:**
   - Go to: `http://localhost:3001/kingdom`
   - Login with admin credentials
   - Click "Queen AI" category in navigation
